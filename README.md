# PimitOS - Rehber
Aktif Mobil İşletim Sistemleri Karşılaştırmalı Veri.

PimitOS - Rehber, mobil işletim sistemleri dünyasındaki güncel pazar paylarını, gizlilik odaklı custom ROM'ları, Linux tabanlı açık kaynak projeleri ve tarihsel sistemleri tek bir modern sayfada sunan bağımsız bir web rehberidir.
Proje tamamen statik (HTML + CSS + JS) yapıdadır; herhangi bir backend, framework veya derleme adımına ihtiyaç duymaz. Tarayıcıda doğrudan çalışır ve GitHub Pages, Netlify veya Vercel gibi platformlarda saniyeler içinde yayınlanabilir.

Tema Yönetimi
Karanlık/Açık mod geçişi, localStorage ile kalıcı tercih, T tuşu kısayolu
📱 Tam Responsive
Mobil-first yaklaşım, 480px / 968px breakpoint'ler, esnek grid yapısı
🌌 Animasyonlu Arka Plan
CSS tabanlı sonsuz yıldız/uzay efekti, blur glow orb, düşük GPU tüketimi
🎨 Tipografi & İkonlar
Google Fonts (Ubuntu, Poppins, Inter) + Bootstrap Icons CDN
📊 Yapılandırılmış Veri
Durum badge'leri, hover efektli tablolar, senaryo tabanlı kartlar
⌨️ Erişilebilirlik
ARIA etiketleri, klavye navigasyonu, focus-visible stilleri
Sıfır Bağımlılık
Tek dosya deploy, CDN optimizasyonu, < 50KB toplam boyut

🛠️ Teknolojiler
Frontend:
  - HTML5 (Semantik etiketler, ARIA)
  - CSS3 (Custom Properties, Grid, Flexbox, Animations)
  - JavaScript ES6+ (DOM, Event Delegation, IntersectionObserver)

External:
  - Bootstrap Icons: https://icons.getbootstrap.com/
  - Google Fonts: Ubuntu, Poppins, Inter
  - Hosting: GitHub Pages / Netlify / Vercel (Opsiyonel)

🚀 Hızlı Başlangıç
1️⃣ Doğrudan Çalıştırma
# Repoyu klonlayın
<code>git clone https://github.com/kullaniciadi/OSRehber.git</code>
<code>cd OSRehber</code>

# index.html dosyasını tarayıcınızda açın
# veya yerel sunucu ile çalıştırın:
<code>python -m http.server 8000   # Python</code>
# veya
<code>npx serve .                  # Node.js</code>

2️⃣ Yayın (GitHub Pages)
Depo ayarlarından Pages bölümüne gidin
Branch: <code>main, Folder: / (root)</code> seçin
URL otomatik oluşturulur: <code>https://kullaniciadi.github.io/pimios</code>
